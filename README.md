# 南台科技大學資工系實務專題 - 自習室空位即時偵測系統

## 1. 專案簡介
本專案旨在利用電腦視覺技術 (YOLOv8) 解決校園自習室座位資訊不透明及佔位問題。

## 2. 核心功能
* **即時偵測**：自動辨識座位是否有人使用。
* **佔位判定**：偵測座位上是否有物品但長期無人，並進行標記。
* **視覺化介面**：透過網頁地圖顯示即時空位分布。

## 3. 技術架構簡述
* **AI Model**: YOLOv8
* **Backend**: Python (Flask)
* **Database**: Firebase Realtime Database
* **Frontend**: HTML5 / JavaScript (RWD)
