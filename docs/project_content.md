# 專題內容說明：智慧校園自習室空位即時偵測系統

## 一、 研究動機與背景
在南台科技大學校園中，自習室與圖書館是學生學習的核心場所。然而，現行管理模式存在以下痛點：
1. **資訊不透明**：學生無法預知是否有空位，常導致白跑一趟。
2. **佔位問題嚴重**：許多座位雖無人使用，卻因放置個人物品導致他人無法使用。
3. **管理成本高**：依靠人工巡檢維護秩序效率低下。

## 二、 研究目的
本計畫旨在開發一套基於電腦視覺 (Computer Vision) 的即時監測系統，達成以下目標：
* 提供即時座位狀態查詢。
* 自動辨識並標記「疑似佔位」行為。
* 產出校園空間使用率分析報表。

## 三、 系統功能定義
1. **即時空位偵測**：辨識每個座位為「空閒」、「使用中」或「佔位」。
2. **視覺化地圖**：以 RWD 網頁呈現自習室平面圖，並以顏色標記狀態。
3. **自動更新機制**：每隔固定時間（如 30 秒）重新整理狀態。
4. **隱私處理**：所有影像僅進行在地化特徵提取，不存儲人臉影像。

## 四、 預期成果
* 完成一套可運行的系統原型 (MVP)。
* 準確率達 90% 以上的偵測模型。
* 提升校園公共資源的使用效率，減少學生尋找座位的時間成本。
